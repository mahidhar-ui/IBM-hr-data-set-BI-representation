# IBM-hr-data-set-BI-representation
real time IBM hr data power BI dashboard
HR Analytics Dashboard 📊
Project Overview

This project is an HR Analytics Dashboard built using Microsoft Power BI to analyze employee data and identify key factors affecting employee attrition.

The dashboard helps HR teams and business managers make data-driven decisions to improve employee retention, workforce planning, and organizational performance.

Objectives
Analyze employee attrition patterns.
Identify departments with high attrition.
Understand the impact of salary, overtime, and job role on employee turnover.
Track employee demographics and performance metrics.
Support HR in creating retention strategies.

Key Insights

Some important insights generated from the dashboard:

Employees doing overtime show higher attrition.
Lower salary employees are more likely to leave.
Younger employees have a higher turnover rate.
Sales and technical departments show higher attrition.
Employees with poor work-life balance are more likely to resign.
